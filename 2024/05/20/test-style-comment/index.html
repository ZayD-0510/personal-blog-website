九、Valine 评论功能测试

文章底部应显示 Valine 评论区，请进行以下测试：

  1. 检查评论框是否正常显示（包含昵称、邮箱、网址输入框和提交按钮）；
  2. 尝试提交一条测试评论（如“测试评论1：样式和评论功能正常！”）；
  3. 提交后刷新页面，检查评论是否在列表中显示；
  4. 若配置了头像，检查评论者头像是否正常加载；
  5. 若开启了“访问量统计”，检查文章顶部/底部是否显示访问次数。

如果评论区未显示，请参考之前的排查步骤：检查 Valine 依赖安装、LeanCloud 凭证配置、安全域名添加，以及浏览器控制台是否有报错。

测试完成后：若所有样式（标题、列表、代码块、图片等）排版正常，且评论能提交并显示，说明你的 Hexo 文章样式和 Valine 评论功能已配置成功！
